{{/* Renders an HTML picture element containing one source for every file in the page bundle matching the passed stem name. If the stem is "FOO", the sources would be "FOO.avif", "FOO.webp", and "FOO.jpg" -- assuming all three of these files existed in the page's content directory. The `pictureFallbackTypes` site parameter controls which media types are used for browsers that don't understand the picture element and/or the newer file formats it brings. The first fallback type that matches a file will be placed in an img tag at the end of the shortcode's output. Named parameters: stem: The stem of the audio file(s) to include in the source list. alt (optional): If non-empty, added as an alt="..." attribute. If this shortcode instance is nested within something that has a `caption` parameter (e.g. %figure%) that is non-empty, that value is used as a default when this is empty. class (optional): If non-empty, added as a class="..." attribute. */}} {{- $stem := .Get "stem" }} {{- $alt := .Get "alt" }} {{- $class := .Get "class" }} {{- $matches := printf "%s.*" $stem | .Page.Resources.Match }} {{- $fallback := site.Params.shortcode.pictureFallbackTypes }} {{- with .Parent }} {{- with .Get "caption" }} {{- $alt = $alt | default . }} {{- end }} {{- end }} {{- .Page.Scratch.Add "shortcodesPictureIndex" 1 }} {{- $index := .Page.Scratch.Get "shortcodesPictureIndex" }} {{- $loading := cond (gt $index site.Params.shortcode.pictureEagerMaximum) "lazy" "eager" }} {{- "" }} {{- range $matches }} {{- "" }} {{- end }} {{- with index (where $matches "MediaType.Type" "in" $fallback) 0 }} {{- "" }} {{- end }} {{- "" }}{{ "" -}}
